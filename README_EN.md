# CSS Carnival - Midnight Neon Carnival

English | [中文](README.md)

A retro‑futuristic amusement park scene built with CSS animation and a touch of JavaScript. It features a neon hero sign, layered atmospheric effects, and four animated rides: ferris wheel, roller coaster, carousel, and bumper cars.

## Highlights

- Night and day themes with a single toggle
- Glassmorphism control dock: play, pause, reset, and speed
- CSS variables drive palette and animation speed
- Scroll‑reveal entrance animations
- Lightweight toast notifications and an Easter egg
- Responsive layout with reduced‑motion support

## Rides

- Ferris wheel with level cabins
- Roller coaster driven by SVG path + `offset-path`
- Carousel with rotating platform and bouncing horses
- Bumper cars with multi‑car paths and spark effects

## Controls

- Play: start all animations
- Pause: pause all animations
- Reset: quick animation restart
- Theme: toggle night/day
- Speed: slider adjusts overall animation timing

## Keyboard Shortcuts

- `Space` toggle play/pause
- `R` reset animations
- `T` toggle theme

## Tech Stack

- HTML + CSS + JavaScript
- CSS animations, filters, gradients, shadows, and masks
- `offset-path`, `conic-gradient`, `backdrop-filter`
- Theme and speed controlled via CSS custom properties

## Usage

1. Clone or download the repo
2. Open `index.html` in a modern browser

## Customization

- Tweak colors and timings in `styles.css`
- Adjust interaction behavior in `script.js`

## Notes

- Modern browsers recommended for the best effects
- Reduced‑motion preference simplifies animations
