# CSS Carnival - 午夜霓虹嘉年华

[English](README_EN.md) | 中文

一个复古未来感的纯前端动画场景，主题为“午夜霓虹嘉年华”。通过 CSS 动画与少量 JavaScript 交互，呈现摩天轮、过山车、旋转木马与碰碰车等游乐设施，并配合霓虹标题、扫描线、噪点纹理与粒子漂浮等氛围效果。

## 特色亮点

- 霓虹夜景与日间主题一键切换
- 玻璃质感控制台：播放、暂停、重置与速度调节
- CSS 变量驱动的配色与动画速度
- 滚动进入动画（IntersectionObserver）
- 轻量交互提示（Toast）与彩蛋效果
- 自适应布局与减少动态偏好支持

## 场景与设施

- 摩天轮：旋转轮盘与保持水平的座舱
- 过山车：SVG 轨道 + `offset-path` 运动
- 旋转木马：旋转平台、上下起伏与圆锥顶棚
- 碰碰车：多车位移轨迹与闪光火花

## 交互控制

- 播放：启动所有动画
- 暂停：暂停所有动画
- 重置：快速重启动画
- 主题：夜景/日景切换
- 速度：滑块调整整体动画节奏

## 键盘快捷键

- `Space` 切换播放/暂停
- `R` 重置动画
- `T` 切换主题

## 技术实现

- HTML + CSS + JavaScript
- CSS 动画、滤镜、渐变、阴影与遮罩
- `offset-path`、`conic-gradient` 与 `backdrop-filter`
- 主题与速度基于 CSS 变量统一控制

## 使用方式

1. 克隆或下载仓库
2. 直接打开 `index.html`

## 自定义

- 在 `styles.css` 中调整配色与动画参数
- 在 `script.js` 中修改交互逻辑或提示文案

## 注意事项

- 建议使用现代浏览器以获得最佳效果
- 若启用“减少动态效果”，动画会被简化
